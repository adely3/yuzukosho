  • cntl+E 行の末尾に移動
  • cntl+K カーソルより右側前消去
  • ↑html→CSSのclass指定に有用
  • cmd+shift+F フォルダ内検索
  • cmd+P クイックオープン
  • cntl+G 指定した行にジャンプ
  • shift+opt+f 整形

WP基礎概念（wordpress codex）

  • 基礎概念1 テーマの変更・カスタマイズ
    →オリジナルのテーマや配布されているテーマを適用できる
  • 基礎概念2 データベースとテンプレートタグ
    →データベースからデータをPHPで抽出・出力している
  • 基礎概念3 CMS機能:投稿と固定ページ
    →投稿：新着情報やブログなど 固定ページ それ以外
  • 基礎概念4 テンプレートファイル
    →外観→テーマエディターでPHPの編集画面が出るテンプレート階層
    読み込むテンプレートによって同じ内容でも異なった表示をすることができる
    ↓主なテンプレートファイル
    • front-page.php:トップページ
    • page.php:固定ページ
    • single.php:投稿ページ
    • archive.php:記事一覧
  • 基礎概念5 インクルードタグ（共通パーツ）
    →共通部（ヘッター・フッター・サイドバー）などはインクルードしている
  • 基礎概念6 プラグイン（拡張機能）
    →最終更新・バージョンとの互換性・有効インストール数を要チェック →例:Contact Form 7(有名らしい)
  • WP頻出PHP3種類
    • 条件分岐 if elseif else switch case: break default:条件分岐タグ
    • テンプレートタグ・パラメータ
    • ループ if while
